String conversion via perl

The following code converts and maps ASCII character strings into a hexadecimal ASCII code value array.

The "convert" subroutine is capable for the conversion.
"convert" executes as the following to perform its task.
1. The first argument is passed to a variable "$string".
2. The string is then converted to a decimal value with "unpack".
3. Unpacked values are mapped to an array with hexadecimal conversion with "sprintf".
4. The array containing hexadecimal values is returned. 

#!/usr/bin/env perl

sub convert {
  my $string = shift;
  my @array = map{sprintf "%x", $_}unpack("C*", $string);
  return @array;
}
